Group-photo
===========

![Screenshot](https://github.com/glenoir62/Avalon-announcements/blob/master/readme.png?raw=true)

Group-photo lists the people of Liferay portal.

Ability to search on :

- lastname;
- firstname;
- email address;
- screen name;
- keywords.

Powered By Liferay, Alloyui, Bootstrap.
